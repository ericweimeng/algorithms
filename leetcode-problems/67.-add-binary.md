# 67. Add Binary

> Given two binary strings, return their sum \(also a binary string\).
>
> The input strings are both **non-empty** and contains only characters `1` or `0`.
>
> **Example 1:**
>
> ```text
> Input: a = "11", b = "1"
> Output: "100"
> ```
>
> **Example 2:**
>
> ```text
> Input: a = "1010", b = "1011"
> Output: "10101"
> ```

## Solutions

### Approach \#1 Optimal solution

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        A = list(a)
        B = list(b)
        carry = 0
        res = ''
        while A or B or carry:
            if A:
                carry += int(A.pop())
            if B:
                carry += int(B.pop())
            res += str(carry % 2)
            carry //= 2          
        return res[::-1]
```
